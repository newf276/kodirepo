
In Settings-->Features-->Custom Actions there are a few settings you can try that will attempt to make various Newf specific adjustments to default Kodi behavior.

- "Open Newf Menu Editor Instead of Kodi Context Menu (Menus)": When enabled, this will open Newf's "Menu Editor" feature when the context menu is used on Root/Movies/TV Shows menus.
- "Open Newf Options Instead of Kodi Context Menu (Media)": When enabled, this will open Newf's "Options" feature when the context menu is used on media within Newf.
- "Open Newf Extras Instead of Kodi Info Dialog": When enabled, this will open Newf's "Extras" feature when "Information" is called on media within Newf.

These are semi-experimental features that may not work well on certain skins/hardware.
